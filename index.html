A little bit about me

I'm passionate about coding.

I'm Laura Faithfull and have a love for hills and adventure. My adventure into code started during Covid-19, when I was no longer able to work as an Osteopath during lockdown. Having worked in manual therapy for over a decade, computers weren't a regular part of life, but learning to code was exciting.

Coding is an addiction to order and clean lines. To create a website and see order from potential chaos is superb. To see a sleek website that is easy to use and looks good brings inspiration.
